### Node.js 脚本 (.mjs)
1. extract_thesis_refs.mjs - 提取和检查论文参考文献
   
   - 从 Word 文档 XML 中提取文本内容
   - 查找"参考文献"位置并显示文档末尾部分
2. inspect_docx_drawings_context.mjs - 检查文档中图片的上下文
   
   - 提取每个 <w:drawing> 图片元素附近的文本
   - 显示每个图片前的最后几段文本内容
3. inspect_docx_media.mjs - 检查文档中的媒体文件
   
   - 解析 Word 文档的关系文件，建立 ID 到媒体文件名的映射
   - 按出现顺序列出所有嵌入的图片文件
4. map_docx_figures.mjs - 映射文档中的图编号
   
   - 检查图3.1-3.3、图4.1-4.4、图C-1-8的位置
   - 查找每个图片嵌入点前最近的图编号标题
5. patch_docx_document_xml.mjs - 直接修改 Word 文档 XML
   
   - 批量替换占位文本（如图标题）
   - 添加"稿面字数与页数核对说明"
   - 自动备份原文件
6. replace_docx_plantuml_media.mjs - 替换文档中的 PlantUML 图表
   
   - 将 thesis_assets/plantuml/out 下的 PNG 图替换入 Word
   - 按图序映射：image1-7 对应图3.1-3.3、图4.1-4.4
7. svg_to_png_for_word.mjs - SVG 转 PNG（Word 兼容）
   
   - 将 thesis_assets 下的 SVG 转换为 PNG
   - 使用 sharp 库，设置合适的尺寸和 DPI
### PowerShell 脚本 (.ps1)
8. insert_plantuml_figures_to_doc.ps1 - 通过 Word COM 插入图表（.doc 格式）
   
   - 使用 Word COM 对象自动化操作
   - 在指定锚点文本后插入 PlantUML 图表和题注
   - 自动居中并调整尺寸
9. patch_desktop_docx.ps1 - Word COM 方式修补文档（.docx 格式）
   
   - 替换占位文本
   - 插入所有图表（正文图 + 附录C界面截图）
   - 需要先运行 svg_to_png_for_word.mjs 生成 PNG 资源
